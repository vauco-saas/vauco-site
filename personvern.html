Personvernerklæring

Sist oppdatert: 4. juni 2026

Denne personvernerklæringen forklarer hvordan Vauco AS («Vauco», «vi») behandler personopplysninger. Vi behandler opplysninger i tråd med personvernforordningen (GDPR) og norsk personvernlovgivning.

Behandlingsansvarlig
Vauco AS · Org.nr 935 989 779 · Oslo, Norge
Kontakt: chris.christiansen@vauco.no

1. Hvilke opplysninger vi samler inn

  • Kontaktopplysninger du selv oppgir når du sender oss e-post (navn, e-postadresse, innhold i henvendelsen).
  • Kundeopplysninger som er nødvendige for å levere våre moduler, for eksempel konto-e-post og prosjektidentifikatorer hos skyleverandører.
  • Tekniske driftsdata knyttet til bruk av modulene (for eksempel kostnads- og forbruksdata fra din egen skykonto), som behandles for å levere tjenesten.

Nettstedet vauco.no bruker ingen sporingspiksler eller analyseverktøy ved lansering.

2. Formål og rettslig grunnlag

  • Besvare henvendelser — grunnlag: berettiget interesse / samtykke ved at du kontakter oss.
  • Levere og drifte moduler (f.eks. CostGuard) — grunnlag: oppfyllelse av avtale.
  • Overholde lovpålagte krav — grunnlag: rettslig forpliktelse.

3. Hvordan vi beskytter dataene

Modulene driftes på Google Cloud med isolerte kundeinstanser og streng tilgangskontroll. Tilgang er begrenset til autoriserte Vauco-kontoer via OAuth og skybasert tilgangsstyring. Vi samler ikke inn mer enn det som er nødvendig for å levere tjenesten.

4. Databehandlere og lagring

Vi benytter Google Cloud Platform som underleverandør for drift. Personopplysninger lagres ikke lenger enn nødvendig for formålet de ble samlet inn for, eller så lenge loven krever.

5. Dine rettigheter

Du har rett til innsyn, retting, sletting og dataportabilitet, samt rett til å protestere mot behandling og til å klage til Datatilsynet. Henvendelser rettes til e-postadressen over.

6. Endringer

Vi kan oppdatere denne erklæringen. Vesentlige endringer publiseres på denne siden med ny dato.
